## Chalenge Application to convert .txt files into Json format

1. Create a Virtual Environment and Activate: 

1.1 - Unix Systems
```bash
$ virtualenv -p python3 env
$ env/bin/activate
```

1.2 - Windows Systems
```bash
$ python3 -m venv env
$ .\env\Scripts\Activate
```

3. Run Application to convert file:
```bash
$ python main.py
```
